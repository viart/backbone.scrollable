Backbone.Scrollable
===================

Implementation of a scrollable View for Mobile devices.

## Features
* Accelerated scrolling / zooming (see more: http://zynga.github.com/scroller/)
* Sticky headers

## Dependencies
* [Backbone.js][1]
* [Zynga Scroller][2]
